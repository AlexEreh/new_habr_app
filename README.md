# habr app

Habr application.

Main features:
* Cache articles
* Filter articles
* Visual customizations

# Visual customization

* light and dark themes of the app \
  select of a specific mode:
  * constant (dark or light)
  * system theme
  * from time to time
* font size
* text alignment
* line spacing
* code styles \
  select of a specific mode:
  * dark mode coloring
  * light mode coloring
  * app used theme
